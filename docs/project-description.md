## Project Overview

### Project Name
AI Weather Art

### Brief Description
[2-3 sentence summary of what this project is and what it does]
An AI agent that generates art based on the current weather conditions in the user's location.

### Project Goals
* Setup an strands llama AI agent 
* Enable the AI agent to query the current weather conditions for the users' location
* Create a p5.js scene that can take in a declarative json that describes the scene to render.
* Enable the AI agent to understand and produce that declarative json
* Create a simple web ui that has a page with an input field to take in the user prompt and then renders the resulting p5.js scene below.
